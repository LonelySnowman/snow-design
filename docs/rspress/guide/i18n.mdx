# 国际化

支持国际化，使用 `ConfigerProvier` 组件变更语言，安装 `@snow-design/locale` 选择语言包，目前支持 `zh_CN`、`en_US`。

```bash
npm install @snow-design/locale
```

## 基本使用

```tsx preview
import { Pagination, ConfigProvider } from '@snow-design/components';
import { zh_CN } from '@snow-design/locale';

export default () => {
    return (
        <div className="snow-flex">
            <ConfigProvider locale={zh_CN}>
                <Pagination
                    showTotal
                    total={20}
                    pageSize={2}
                    onChange={(a, b) => {
                        console.log(a, b);
                    }}
                />
            </ConfigProvider>
            <Pagination
                showTotal
                total={20}
                pageSize={2}
                onChange={(a, b) => {
                    console.log(a, b);
                }}
            />
        </div>
    );
};
```
