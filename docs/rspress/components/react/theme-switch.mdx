# ThemeSwitch 主题切换器

用于切换应用的主题模式。

## 基础使用

ThemeSwitch 支持传入默认主题状态。

```tsx
import { ThemeSwitch } from '@snow-design/components';

export default () => {
    return (
        <div className="snow-flex">
            <ThemeSwitch defaultThemeType="dark" />
        </div>
    );
};
```

## 受控模式

在受控模式下你可以通过外部值控制主题切换器的状态。

```tsx
import React, { useState } from 'react';
import { ThemeSwitch, Button } from '@snow-design/components';

const Control = () => {
    const [themeType, setThemeType] = useState('light');
    return (
        <>
            <h3>受控模式</h3>
            <ThemeSwitch
                themeType={themeType}
                onChange={(curTheme) => {
                    setThemeType(curTheme);
                }}
            />
            <h4>自定义受控按钮</h4>
            <Button style={{ marginRight: '4px' }} onClick={() => setThemeType('light')}>
                亮色主题
            </Button>
            <Button onClick={() => setThemeType('dark')}>暗色主题</Button>
        </>
    );
};

export default Control;
```

## API 参数

### ThemeSwitch

| 属性             | 说明                 | 类型              | 默认值  |
| ---------------- | -------------------- | ----------------- | ------- |
| onChange         | 主题变更时产生的回调 | () => void        | -       |
| themeType        | 当前主题类型         | 'light' \| 'dark' | 'light' |
| defaultThemeType | 默认主题类型         | 'light' \| 'dark' | 'light' |
| size             | 组件 fontSize 大小   | number            | 1       |
