# ImgLazyLoad 图片懒加载

用于延迟加载图片，提升页面性能。

## 基础使用

ImgLazyLoad 支持通过插槽设置占位内容。

```vue
<template>
    <div style="height: 150vh; padding: 20px;">
        <p>向下滚动查看懒加载图片</p>
        <div style="margin-top: 1000px;">
            <ImgLazyLoad
                src="https://fastly.picsum.photos/id/83/200/300.jpg?hmac=avqtE9ZSAkPbFtYCXzxg4TeAA-fMWqX6jUQeWI_HjLc"
            >
                <template #placeholder>
                    <div
                        style="
                            width: 200px;
                            height: 300px;
                            background: #eee;
                            display: flex;
                            align-items: center;
                            justify-content: center;
                            color: #999;
                        "
                    >
                        Loading...
                    </div>
                </template>
            </ImgLazyLoad>
        </div>
    </div>
</template>

<script setup lang="ts">
import { ImgLazyLoad } from '@snow-design/vue3';
</script>
```

## API 参数

### ImgLazyLoad

| 属性      | 说明         | 类型   | 默认值 |
| --------- | ------------ | ------ | ------ |
| src       | 图片地址     | string | -      |
| threshold | 触发加载阈值 | number | 0      |

### 插槽 Slot

| 属性             | 说明                     | 类型    | 默认值 |
| ---------------- | ------------------------ | ------- | ------ |
| placeholder      | 占位内容                 | VueNode | -      |
| errorPlaceholder | 图片加载后展示的占位内容 | VueNode | -      |
